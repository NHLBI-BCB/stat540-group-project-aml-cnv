Boxplotter for Differential expression analysis for trisomy 8, deletion 5 and deletion 7

This code is for plotting hits of interest from the DEA of the 3key deletion genotypes (trisomy 8, del 5 and del 7). It performs differential expression analysis on cleaned RNA-seq read count data, and when given abrreviated hit names, it finds the full names, prepares the appropriate data matrix for plotting and then boxplots them.
